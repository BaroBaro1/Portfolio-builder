# Portfolio Builder

A modern SaaS platform that enables developers and professionals to build, manage, and publish beautiful online portfolios without writing code.

---

## ✨ Overview

Portfolio Builder is a full-stack web application built with Laravel and React.

Users can create a professional portfolio, showcase their projects, skills, work experience, certificates, and social links through a clean public page.

The project focuses on simplicity, performance, and maintainability.
